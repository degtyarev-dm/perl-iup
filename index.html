Perl module IUP

IUP module is a perl binding for cross-platform GUI toolkit based on the following libraries delivered by Tecgraf - Computer Graphics Technology Group:
  • IUP (portable user interface)
  • CD (canvas draw, a 2D graphics library)
  • IM (image representation, storage, capture and processing)

Download

You can download this project in either zip or tar.gz formats.

You can also clone the project from Git repository by running: 

$ git clone git://github.com/kmx/perl-iup

Basic installation steps are described in README.

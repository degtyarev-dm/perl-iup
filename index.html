perl-iup by kmx

Perl bindings for cross-platform GUI toolkit based on the following libraries:
  • iup library - see http://www.tecgraf.puc-rio.br/iup/
  • im library - see http://www.tecgraf.puc-rio.br/im/
  • cd library - see http://www.tecgraf.puc-rio.br/cd/
